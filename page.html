    Message Template Placeholders

    Enhance your message with dynamic content using these placeholders:

    • {originConnectionName}: The full name of the connection that you want to get an intro from
    • {profileName}: The first name of the person you're messaging (the mutual connection)
    • {fullName}: The full name of the person you're messaging (the mutual connection)
    • {companyName}: The current company name of the connection you want to get an intro from
    • {jobTitle}: The current job title of the connection you want to get an intro from
    • {connectionLinkedInUrl}: The full LinkedIn profile URL of the connection you want to get an intro from

    Note: Ensure your message complies with LinkedIn's terms of service and maintains a professional tone. Personalize the message by adding specific details relevant to each recipient.
